# PROJECT STATUS (Tempest 会員管理システム)

## プロジェクト概要
Tempest店舗向けのWeb会員管理システム。Cloudflare Pages + D1データベースで稼働中。2026-07-01より本格運用開始。

## 現在のステータス
* **ポイント交換カタログ機能**: **本番環境へ反映完了 (2026-07-01)**
  * マイページでのカタログ閲覧（モーダル表示・XSS対策）、管理画面からのインライン追加・編集・無効化に対応。
  * 本番D1に `point_catalog` テーブルおよびショップ・表示順インデックスを追加済み。
* **会員メモ欄・来店取消（-1）機能**: **本番環境へ反映完了 (2026-07-01)**
  * 管理者による会員ごとのメモ機能、来店回数とポイントを連動して差し引く「来店-1」機能を実装。
  * スマホ（iPhone）幅での横スクロールはみ出し防止、タップサイズ確保、ボタン文言のレスポンシブ最適化を適用。
  * 本番D1の `members` テーブルに `admin_memo TEXT` カラムを追加済み。
* **お知らせ・クーポン本文の改行表示対応**: **本番環境へ反映完了 (2026-06-25)**
  * iPhone等で入力された改行を会員マイページおよび管理画面上で正しく改行表示（`white-space: pre-line`）するようUIを調整。
* **会員削除機能（論理削除）**: **本番環境へ反映完了 (2026-06-25)**
  * 本番D1データベースに `deleted_at TEXT` カラムを追加し、論理削除に対応。
